# 要件定義｜Astra

## アプリ名（仮）
Astra Anime Reviews

## 概要
アニメ作品ごとに、ユーザーがレビュー・評価を投稿できる口コミサービス。
将来的にアニメ作品単位のチャット機能を追加できる設計とするが、
MVPではレビュー機能のみを実装対象とする。

## 目的
- アニメ視聴前に、他人の感想・評価を簡単に確認できるようにする
- SNSよりも作品単位で情報が整理されたレビュー体験を提供する
- 個人開発として、実運用可能なWebアプリを構築する

## 想定ユーザー
- アニメ視聴者
- 10代後半〜30代
- スマートフォン中心で利用
- Googleアカウントを所持しているユーザー

## 提供価値
- 作品ごとにレビューが集約されている
- Googleログインのみで簡単に参加できる
- 将来、同じ作品を見ているユーザー同士でチャットできる拡張性

## 機能要件（MVP）
- アニメ作品一覧を閲覧できる
- アニメ詳細ページを閲覧できる
- Googleログインができる
- レビューを投稿できる
- レビュー一覧を閲覧できる

## 将来的に追加予定の機能（MVP外）
- アニメ作品ごとのチャット機能
- いいね機能
- 通報機能
- 通知機能

## やらないこと（MVPでは実装しない）
- チャット機能
- フォロー機能
- DM機能
- 画像アップロード機能

## 非機能要件
- PC / スマートフォン両対応（レスポンシブ）
- SEOを考慮したページ構成
- 無料プラン内での運用を前提とする

## 技術スタック（前提条件）
- Frontend: Next.js（App Router）
- UI: Chakra UI
- Backend: Next.js Route Handler
- Auth: Supabase Auth（Google）
- DB: Supabase
- Hosting: Cloudflare Pages
