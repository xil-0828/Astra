# Astra

アニメ作品の情報・配信状況・レビューをまとめて閲覧できる
**個人開発のアニメ情報・口コミWebアプリ**です。

---

## 主な機能
- アニメ一覧表示（カードUI）
- レビュー・評価機能
- 検索機能
- Googleログイン（Supabase）

---

## 使用技術
- Next.js / TypeScript
- Chakra UI
- Supabase（Auth / DB）
- Cloudflare Pages
- Figma（Auto Layout）

---

## 開発メモ
- Figmaはレイアウト設計のみ
- 動き・スライダーなどは実装側で制御

---

## セットアップ

```bash
npm install
npm run dev
```
