

# Astra

### アニメのレビューや評価、配信状況をまとめて見られるWebアプリ

Astra は、アニメ作品について  
「どんな評価なのか」「どこで見られるのか」を  
ひとまとめで確認できる Web アプリです。

## なぜ作ったか

アニメを見る前や見終わったあとに、  
**他の人がどんな感想を持っているのか、評価はどんな感じなのか**  
を気軽に知りたいと思ったのがきっかけです。

また、掲示板やSNSほど重くなく、  
**同じアニメを見ている人同士で、もっと気軽に感想を共有したり、  
チャットできる場所があったらいいな**と思い、このアプリを作りました。


---

## できること

- **アニメ作品の情報を確認**
  - 作品ごとの基本情報を一覧・詳細で表示

- **レビューを見る／書く**
  - ログインするとレビュー投稿ができます
  - 他の人の感想もすぐに確認できます

- **配信状況のチェック**
  - 配信しているサービスがあるかを表示

- **Googleログイン**
  - Googleアカウントで簡単にログインできます
